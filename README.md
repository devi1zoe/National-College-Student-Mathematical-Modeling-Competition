# 一.AHP层次分析法

- 解决评价类问题(择优，评选最好的方案/员工)
- 解题思路：
    - 根据现有论文，寻找评价的打分指标（外貌、身高..）
    - 确定指标的重要性程度，即权重（三种方法都用）
        - 互联网搜索，给出打分（两者总为1）
        - 将权重两两比较，构成判断矩阵
        - 在求权重前要 进行一致性检验（CR＜0.1）
        - 求占比，处理得权重
        - 多列权重取平均得综合权重
    - 对评价的对象就指标进行打分（然后归一化处理）
        - 在某个指标下，将对象两两比较，构成判断矩阵
        - 同上

# 二.Topsis模型

- 解决评价类问题
- 解题思路：
    - 指标正向化
    - 标准化处理去量纲
    - 熵权法给指标确定权重
        - 计算概率
        - 计算信息熵（归一化）
    - 求优劣距离（多个指标就多个坐标求欧式距离）
    - 得分公式得出后需要归一化

# 三.插值算法

- 现有数据少需模拟产生新值或用于短期预测
- 解题思路：
    - 分段三次埃尔米特插值
    - 三次样条插值

# 四.拟合算法

- 寻找一个函数使其与已有的数据点最为接近，与三功能差不多
- 解题思路：
    - 最小二乘法，定义样本点与拟合曲线的接近程度
    - 用拟合优度评价拟合对参数线性的函数的好坏，非线性函数用SSE误差平方和
    - 用curve fitting曲线拟合工具箱
    - 导出代码后加坐标轴标签
    - 导出图片记得调高分辨率
    - 最后可以模拟数据对拟合结果进行检验

# 五.相关系数

- 用于衡量两个变量之间相关性的大小
- 解题思路：
    - 皮尔逊相关系数(连续数据 正态分布 线性相关)或斯皮尔曼相关系数(两个定序数据之间和其他数据)
        - 题目给了数据，先做描述性统计
            - MATLAB
            - Excel
            - SPSS
            - Stata
        - 绘制散点图可以很清楚的看出相关性，看是否线性分布SPSS
        - 判断数据是否正态分布
        - 相关数据表可视化Excel
        - 对皮尔逊相关系数进行假设检验SPSS（t检验是基于数据正态分布）
            - 对数据进行正态分布JB检验或shapiro-wilk检验或Q-Q图
            - 提出假设
            - 假设成立的条件下构造分布，画出该分布的pdf，给定置信区间，画出接受域和拒绝域
            - 代入相关系数得一个检验值
            - 检验值对应的p值与0.05进行比较（95%）

# 六.典型相关分析

- 研究两组变量之间相关关系（将每组变量中的多个指标线性组合）
- 解题思路：
    - 假设两组数据服从联合正态分布
    - 对两组变量的相关性进行检验
    - 确定典型相关变量的个数，用p值和0.1（灵活处理）比较，小于0.1则99%..
    - 几个典型相关变量，几对典型相关系数
    - 利用标准化后的典型相关变量分析问题

# 七.回归分析

- 研究∑x与y的相关关系，用于解释y的形成机制或预测y。特殊的：研究自变量对y的重要程度（标准化去量纲）——用于识别影响y的重要变量，判断x与y的相关性的方向，由估计权重得到回归系数
- 1. 线性回归
    - 多元线性回归用于处理横截面数据-某一时点收集的不同对象的数据
    - 解题思路：
        - 数据预处理
        - 确定自变量，需要满足外生性，
            - 自变量与扰动项之间有关则存在内生性，有数据的是自变量，无数据的可以当扰动项，内生性会导致回归系数的巨大误差（蒙特卡洛模拟得出）
            - 核心变量与扰动项不相关即可
        - 添加交互项，平方项
        - 自变量中有定性变量，stata可以生成虚拟变量
        - 检验扰动项中是否有异方差-残差图或方差的假设检验-怀特检验
            - 异方差的处理方法：OLS➕稳健的标准误
        - Stata中reg命令即得
        - 检验多重共线性用vif＞10则存在
- 2. 0-1回归
- 3. 定序回归（等级评定）
- 4. 计数回归（泊松分布，每分钟车流量）
- 5. 生存回归（寿命问题）

# 八.图论最短路径问题

- 用于解决最短路径问题
- 画图网站
- 迪杰斯特算法
    - 含有负权重的有向图例外
    - 从一点出发，走最短路径，补充所到点至集合，再从这点出发，走最短路径，重复至每个点都经过

# 九.分类模型

- 用于判别某样东西应该属于哪一类（已知类别）
- 二分类模型
    - SPSS中逻辑回归和Fisher线性判别分析
- 多分类模型
    - SPSS中多分类线性判别和多分类逻辑回归
- 过拟合现象（加入平方项，交互项之后）
    - 解决方法--交叉验证（训练组和测试组）➕多次取平均

# 十.聚类问题

- 用于将多样本分成未知的几种类别（不知能分成几类，不知类别的名字）
- 聚类方法：
    - K-means++聚类（SPSS）（基于距离的聚类）
        - 自定义类别的个数k，好解释即可
        - 随机确定k个初始聚类中心
            - 随机选一个样本作为第一个中心
            - 计算每点到已有中心的最短距离，距离最大的选作中心的概率较大
            - 轮盘法选出下一个中心
            - 重复二三至选出k个中心
        - 计算各点到k个中心的距离，划分至离各点最近的类中
        - 调整中心和新类
        - 循环二三至分类不变
        - 聚类所依靠的数据的量纲不一致时（单位不统一）记得标准化
    - 系统聚类（基于距离的聚类）首选
        - 系统帮忙分类，组数不需自定
            - 用肘部法则从图形中大致
            - 聚合系数折线图（Excel）
        - 将每个对象看成一类，计算两两之间的最小距离
        - 将距离最小的合成一个新类
        - 重新计算新类中心点与所有类的距离
        - 重复三四两步至所有类最后合为一类
    - DBSCAN(基于密度的聚类)
        - 如果只有两个指标，且结果很DBSCAN,不随机的感觉，则用DBSCAN进行聚类

# 十一.时间序列分析

- 用于描述过去，分析规律，预测未来spss
- 时点时间序列和时期时间序列
    - 不可加的为时点序列
- 乘积模型和叠加模型
    - 随着时间推移，波动变大为乘积模型，波动恒定为叠加模型
- 解题思路：
    - 数据预处理
        - 处理数据缺失值SPSS
            - 五种方法
    - 定义时间变量
    - 画时间序列图，判断是否平稳（围绕均值波动，无趋势和季节性）
    - 分析数据是季度数据还是月份数据，观察是否存在季节性波动
    - SPSS专家建模器得最优模型
    - 若结果是ARIMA(P,0,q)可画Acf和Pacf来分析，若结果是ARIMA(P,1,q)可对数据进行画Acf和Pacf来分析，若结果与季节性相关，用时间序列分解
    - 建立时间序列分析模型
        - 季节性分解（要求序列的时间单位＜年）
        - 七种指数平滑方法
            - Simple模型
                - 不含趋势和季节
            - linear trend模型
                - 线性趋势，不含季节
            - Damped trend模型
                - 线性趋势渐弱，不含季节
            - Simple seasonal模型
                - 不含趋势 含简单的季节
            - Winters' additive模型
                - 含线性趋势 含稳定的季节成分
            - Winters' multiplicative模型
                - 含线性趋势 含不稳定的季节成分
        - ARIMA模型（差分方程）
            - ARIMA(0,1,1)
                - 不含趋势和季节
            - ARIMA(0,2,2)
                - 线性趋势，不含季节
            - ARIMA(1,1,2)
                - 线性趋势渐弱，不含季节
            - SARIMA(0,1,1)×(0,1,1)s
                - 不含趋势 含简单的季节成分
            - SARIMA(0,1,0)×(0,1,1)s
                - 含线性趋势 含稳定的季节成分
    - 预测未来数值要结合背景

# 十二.预测模型

- 结合背景，合理假设，多做解释
- 解题思路：
    - 先画时间序列图并简单分析趋势
    - 将数据分为训练组和对照组
    - 用不同模型对训练组来建模，用试验组判断预测效果
    - 得到误差最小的模型后利用全部数据来建模，并对未来数据进行预测
    - 画出预测后数据和原来数据的时序图，看看预测的是否合理
- 灰色预测
    - GM(1,1)
        - 用于以年份度量的非负数据，数据要能经过准指数规律的检验，数据的期数较短且和其他数据的关联不强
- 神经网络
    - MATLAB
    - 论文中找原理

# 十四.主成分分析

- 用于降维，将多个指标转化为少数几个彼此无关的主成分，当问题涉及多变量且变量间存在很强的相关性时，可用
- 用于聚类分析
    - 将数据降维至两维，方便画图
- 用于回归分析
    - 解决完全多重共线性问题（与逐步回归一起用）
- 难点
    - 对选取的主成分（贡献率超过80%）要给出好的解释

# 十五.因子分析模型

- 用于降维，将多个指标转化为少数几个综合因子，比主成分更易得到解释，优于主成分分析
- SPSS
    - 判断是否使用因子分析
        - KMO检验和巴特利特球形检验
    - 确定因子个数
        - 碎石图
    - 因子旋转后的成分矩阵更利于解释

# 十六.岭回归和lasso回归

- lasso回归只起变量筛选作用（一般用OLS回归，计算VIF＞10时用lasso回归，不用逐步回归）
    - 在多元线性回归中的OLS回归模型中加了惩罚项
        - 识别不重要变量，简化模型
        - 让不满足列满秩的模型，变得可估计
            - 处理指标多样本少的情况
        - Stata
        - 最小化均方误差最常用
        - 标准化保证量纲一致
    - lasso回归优于岭回归，将不重要的回归系数压缩至零（缺点：无显式解，使用近似估计算法，解决方法：最小均方误差法）
    - 标准化
        - MATLAB：zscore函数
        - SPSS：分析_描述统计_描述
        - Stata：egen X＝std（x）
    - K折交叉验证法来选择最佳的调整参数
    - 结果可能随seed的变化而变化，剔除的多重共线性变量是相对的

# 十八.Floyed算法

- 解决任意两点间（迪杰斯特拉算法只能计算给定的起点和终点之间）的最短路径问题（无向有向，存在负权重）但不可以存在负权回路

# 十九.ARCH和GARCH模型

- 只适用于扰动项存在条件异方差时
    - 长期来看数据平稳，长期方差定；短期来看方差不稳定
    - 三种方法来判定
- 股票，经济
- ACF,PACF
- AIC,BICA

# 二十.正态分布均值的假设检验

- 第五章的假设检验部分为基础
- Z检验
    - 总体方差已知 均值未知
    - 总体服从正态分布
- t分布
    - 样本方差可求 总体方差和均值未知
    - 总体服从正态分布
- 单主题或检验两个正态分布总体均值差
- 逐对比较法
    - 一个对象，两种方法，看方法要没有差异
- 分布拟合检验
    - 不符合jb检验和Shapiro-wilk检验的，不是正态分布
    - 卡方分布
